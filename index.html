📊 About This Analysis

Data Collection: Data was collected from 2,000 active social media users over a 12-month period (January-December 2024).

Metrics Tracked: Daily active hours, platform preference, posting frequency, engagement rates (likes, comments, shares), and follower growth.

Analysis Methods: Statistical analysis (ANOVA, t-tests), K-means clustering for user segmentation, time series decomposition, and correlation analysis.

Tools Used: Python (Pandas, NumPy, Scikit-learn), R for statistical validation, and Chart.js for visualization.
